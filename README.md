## Password01

password01 项目是一个基于tornado web 开发的密码同步工具。该项目属于练手项目，有问题欢迎随时沟通相互学习。


##### steps:

###### 1、docker pull python:3.8-slim-buster
###### 2、pip install -r requirements.txt
###### 3、sh web.sh


##### ps:

邮箱需要自己申请开通pop3，填写收发地址(send.py)
